# Git

  • # 将本地项目上传到远程仓库操作步骤：

    git init

    git remote add origin 远程仓库地址

    git pull origin master

    git add .

    git commit -m '注释'

    git push origin master

    [注意]：拉取远程仓库代码时，若有同名文件，记得将本地的文件删除（如：README.md）。
